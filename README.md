# classroomsounds — v5 trial stack

The **v5 trial firmware, backend, and dashboards** for the Lyppard
classroom-sounds trial.

This repo is **maintenance mode only**. The v6 commercial product is a
complete rebuild and lives in a separate repo,
`chestnutinfrastructure/holly-platform`. v6 work does not happen here.

Start any Claude Code session on this repo by reading **`CLAUDE.md`** at
the root — it spells out what is and isn't allowed here, the known
quirks (lock contention etc.), and the file map. Carson sessions pick
it up automatically.

For the v6 build, see `holly-platform-seed/UPLOAD_INSTRUCTIONS.md` —
the seed folder holds everything needed to populate the holly-platform
repo and kick off a fresh v6 chat.
